# VVS Assistance+ prototype

Dette projekt er en interaktiv prototype udviklet som en del af mit hovedopgaveprojekt.

Formålet er at udvikle og teste et abonnementsbaseret servicekoncept for VVS Assistance med fokus på brugeroplevelse og kundeloyalitet.

## Teknologier
- HTML
- SCSS
- JavaScript

## Status
Under udvikling - første struktur og layout er opsat.
