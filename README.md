# VVS Assistance+ prototype

Dette projekt er udviklet som en interaktiv prototype til min hovedopgave på multimediedesignuddannelsen.

Projektet tager udgangspunkt i virksomheden VVS Assistance og fokuserer på udviklingen af et abonnementsbaseret servicekoncept med fokus på tryghed, gennemsigtighed og langsigtet kundeloyalitet.

Prototypen er udviklet med fokus på brugeroplevelse, informationsarkitektur og digitale brugerflows. Projektet er baseret på research, kvalitative interviews, UX-metoder og iterative tests.

## Formål

Formålet med projektet var at undersøge:

- hvordan et serviceabonnement kan skabe værdi i VVS-branchen
- hvilke behov og frustrationer brugerne oplever i forbindelse med VVS-services
- hvordan digital kommunikation kan skabe mere tryghed og overskuelighed
- hvordan UX og interaktionsdesign kan understøtte tillid og konvertering

## Funktioner

Prototypen indeholder blandt andet:

- Landing page for VVS Assistance+
- Serviceaftale-flow
- Kontaktflow med formular-feedback
- Responsiv navigation og burger-menu
- Interaktive elementer udviklet med JavaScript
- Think-aloud testforbedringer og usability-iterationer
- Fokus på responsivt design og visuel identitet

## Teknologier

Projektet er udviklet med:

- HTML5
- SCSS
- JavaScript

## UX og metode

Projektet bygger blandt andet på:

- Kvalitative interviews
- Value Proposition Canvas
- Wireframes og mockups
- Think-aloud usability-tests
- Iterativ designproces

## AI-anvendelse

AI blev anvendt som sparringsværktøj til:

- strukturering
- teknisk vejledning
- sproglig sparring

Alle løsninger og beslutninger blev kritisk vurderet og tilpasset projektets egne brugerindsigter og faglige mål.

## Sådan åbnes projektet lokalt

1. Download eller clone projektet
2. Åbn projektmappen i Visual Studio Code
3. Installer extensionen **Live Server**
4. Højreklik på `index.html`
5. Vælg **Open with Live Server**

Projektet åbner herefter i browseren på en lokal server.
